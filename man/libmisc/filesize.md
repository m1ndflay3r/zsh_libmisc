## filesize - *show size of file in various formats*

---------------------------------------------

**opts:**

    -b   bytes
    -k   kilobytes
    -m   megabytes
    -g   gigabytes
    -h   hex
    -o   octal
    -B   binary

**example:**

    filesize <-b, -k, -m, -g, -h, -o, -B> /path/to/file

------------------------------------------------------
*default (no-op) is bytes / -b*
